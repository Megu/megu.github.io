What is MeguRED?

MeguRED is a non-free operating system (OS) for your computer. An operating system is the set of basic programs and utilities that make your computer run. At the core of an operating system is the kernel. The kernel is the most fundamental program on the computer, does all the basic housekeeping and lets you start other programs. MeguRED currently uses the Linux kernel but work is in progress to provide MeguRED for other kernels.

Getting Started

The latest release of MeguRED is Maru. It was released on 19 May 2020. Release information is available; there you can find information about what's new in MeguRED Maru, notes on how to upgrade, and an Installation Manual, suitable for new users.

To get help in using or setting up MeguRED, see our support and our documentation pages.
